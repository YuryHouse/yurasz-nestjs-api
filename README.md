## Description

API created with help [Nest](https://github.com/nestjs/nest) framework TypeScript, PostgreSQL DB and Object-relational mapping (ORM) tool called TypeORM.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
